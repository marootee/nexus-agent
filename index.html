Get Started in Minutes

Simple installation and quick setup

1. Installation

$ pip install nexus-agent pip install nexus-agent

2. Create Your First Agent

agent.py from nexus_agent import Agent, Tool

# Create an agent
agent = Agent(name="MyAgent", model="gpt-4")

# Add tools
@agent.tool
def calculate(expression: str) -> float:
  return eval(expression)

# Run the agent
result = agent.run("Calculate 2 + 2 * 3")
print(result)

3. Deploy

Docker deployment docker run -e AGENT_KEY=your_key nexus-agent:latest
